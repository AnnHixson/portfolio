Kingme.gg offers a way to play checkers on your computer. It is a MERN stack single-page application, and it was a group project. My portion of the project was the entirety of the functionality of the checkers game. The game logic is over 3,000 lines of JavaScript code, and it allows for every legal move in a game of checkers while preventing illegal moves. For a sample guest account, use the email: guest@mail.com, and the password: guest123. To access the checkers game click on "Play Hot-Seat."

Technologies used: MERN Stack and JavaScript.

Roll of the Dice is an online dice roller made using JavaScript, HTML, CSS, and Bootstrap. The user can roll any number of dice with any number of sides. Modifiers and additional dice can be added in the same roll. Results of the individual dice are displayed and tallied for convenience. I designed it with D&D in mind, but it can be used for other dice-based games, or for other dice-rolling purposes.

Technologies used: JavaScript, HTML, CSS, and Bootstrap.
